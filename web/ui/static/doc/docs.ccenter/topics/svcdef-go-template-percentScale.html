percentScale function

Multiply an item by a percentage.

Syntax: percentScale Item Percentage

Example: {{percentScale .RAMCommitment 0.8}}

If the value of RAMCommitment is 2147483648 bytes, the preceding template is replaced by 17179818. The item to transform is rounded and the result is converted to an integer.
